### SmartInfra FloodAlert

## Sobre o projeto

O SmartInfra FloodAlert é um sistema em Java voltado ao monitoramento de enchentes em regiões urbanas. A aplicação simula o uso de sensores para acompanhar o nível da água e o volume de chuva, permitindo identificar situações de risco e emitir alertas.

O projeto está relacionado ao ODS 9 — Indústria, Inovação e Infraestrutura, pois propõe uma solução tecnológica ligada à infraestrutura inteligente, automação e prevenção de problemas urbanos.

---

## Funcionalidades
* Cadastro de sensores de água
* Cadastro de sensores de chuva
* Registro de medições
* Consulta dos sensores cadastrados
* Verificação do nível de risco
* Emissão de alertas

## Principais classes
* Sensor: classe abstrata base para os sensores.
* SensorAgua: representa o sensor responsável pelo nível da água.
* SensorChuva: representa o sensor responsável pelo volume de chuva.
* Alerta: responsável pela emissão dos alertas.
* SistemaMonitoramento: gerencia os sensores cadastrados.
* NivelRisco: define os níveis de risco do sistema.
